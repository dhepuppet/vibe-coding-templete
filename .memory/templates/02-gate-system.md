# Gate 시스템

Gate 미통과 = 다음 기능 금지

---

## 기본 Gate
- **Gate 1**: 기본 흐름 10회 안정
- **Gate 2**: 에러 시 로그 재현 가능
- **Gate 3**: 중단/재시작 정상
- **Gate 4**: 엣지케이스 3개+

---

## 타입별 예시

### 웹앱
□ 로그인→기능→로그아웃 10회
□ 폼 검증 (빈값/특수문자)
□ 새로고침 시 상태 유지

### API
□ 정상 요청 10회
□ 잘못된 입력 → 에러코드
□ 인증 실패 → 401/403

### CLI
□ --help 정상
□ 인자 누락 → 안내
□ 에러 → exit code ≠ 0
